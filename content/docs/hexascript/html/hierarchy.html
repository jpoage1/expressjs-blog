  • Compiler

    • 5 Step Pipeline

      1. Frontend Lex/Parse (JSON AST to `ILanguageProvider` validation).
      2. Seeded Interning (Function/Object to Base 38 Macro mapping).
      3. Linearization (AST to 1D token stream).
      4. Logic Squeeze (SSA reduction, constant folding, DCE).
      5. Tier 0 Emission (.hex binary generation and Shadow Alias Registry mapping).
